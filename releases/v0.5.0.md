# TSMM v0.5.0

## Summary

This release adds a modular extension architecture for emerging application domains while keeping the TSMM core stable.

## Added

- Agentic AI extension with domain abstractions for actions, delegation, capabilities, resources, execution context, oversight, trace, and risk tier
- Verifiable Trust Communities extension with abstractions for community, membership, standing, charter, recognition, boundary rules, sanctions, and remediation
- extensions landing page for GitHub Pages navigation
- extension schemas and worked example instances
- GitHub Pages workflow file aligned with the published site structure

## Updated

- README, docs navigation, and pages landing content to surface the extension strategy
- release and version metadata to v0.5.0
- repository packaging to keep documentation aligned with current version references

## Design position

TSMM now makes the layering rule explicit:

- core for invariants
- extensions for recurring specializations
- crosswalks and profiles for concrete implementation mappings
