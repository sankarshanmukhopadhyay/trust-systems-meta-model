# TSMM v0.3.0

## Summary

This release hardens TSMM after review against TRQP-TSPP, ERC-8004-CSP, DCAS, and the repository documentation freshness checklist.

## Added

- GitHub Pages readiness with `docs/index.md` and `_config.yml`
- documentation governance and freshness metadata
- glossary
- DCAS crosswalk
- explicit modeling for governance context, profile, requirement, and assessment
- expanded schema and refreshed examples
- security reporting guidance

## Impact

TSMM now maps more cleanly to profile-driven and assurance-oriented repositories, not just registry and consumer-signal repositories.
