# TSMM v0.5.0

## Summary

This release moves TSMM from a conceptual reference set toward an operational architecture toolkit.

## Added

- formal entity catalog with mandatory fields and implementation guidance
- normative relationship specification with cardinality guidance
- lifecycle model covering issuance, delegation, suspension, revocation, expiry, remediation, and archival
- TSMM conformance profiles for Minimal, Operational, and Assured implementations
- effect evaluation model for execution-time trust decisions
- reusable trust-system threat taxonomy
- reference architecture patterns for trust registries, delegated agents, credential verification, and assurance evidence
- GitHub Pages deployment workflow through GitHub Actions
- refreshed navigation for docs and pages landing page

## Impact

TSMM now supports stronger portability across architecture design, conformance framing, and evidence-aware governance systems.
